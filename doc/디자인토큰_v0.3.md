# Design Tokens v0.3

> 와이어프레임의 시각 언어를 코드로 옮긴 디자인 시스템 정의. 
> Tailwind config + shadcn/ui CSS 변수 기반.

---

## 🎨 디자인 원칙

1. **모노톤 + 한 점의 액센트** — 흑백 베이스에 SK 레드(#EA002C)를 포인트로
2. **상태는 색으로 명확히** — 가용(초록), 대여중(파랑), 연체(빨강)
3. **본문은 Noto Sans KR, 숫자/코드는 JetBrains Mono** — 데이터가 많은 화면에서 가독성 확보
4. **반경은 작게** — 4px 기준. 카드는 6px까지. 화려한 곡선 지양
5. **그림자는 절제** — 카드 자체엔 거의 없고, 모달·드롭다운에만

## 2. 타이포그래피

### 폰트 패밀리

| 토큰 | 폰트 | 용도 |
|---|---|---|
| `font-sans` | Noto Sans KR | 본문 (한국어 + 영어) |
| `font-mono` | JetBrains Mono | 숫자, 코드, 식별자, 라벨 |
| `font-display` | Noto Sans KR (700, 900) | 큰 제목 |

### 크기 시스템

| 토큰 | px | rem | 용도 |
|---|---|---|---|
| `text-xs` | 11px | 0.6875 | 라벨, 메타 정보 |
| `text-sm` | 12px | 0.75 | 보조 텍스트, 테이블 셀 |
| `text-base` | 13px | 0.8125 | **본문 기본** (밀도 높은 관리자 화면 기준) |
| `text-md` | 14px | 0.875 | 구성원 페이지 본문 |
| `text-lg` | 16px | 1.0 | 강조 본문, 카드 제목 |
| `text-xl` | 18px | 1.125 | 페이지 부제목 |
| `text-2xl` | 22px | 1.375 | 패널 제목 |
| `text-3xl` | 26px | 1.625 | 섹션 제목 |
| `text-4xl` | 32px | 2.0 | 페이지 제목 |
| `text-5xl` | 42px | 2.625 | Hero 제목 |

> **밀도 노트**: 관리자 화면(1920×1080 기준)은 정보 밀도가 높아 본문을 13px로 설정. 구성원 페이지는 14px로 살짝 키워 가독성 우선.

## 3. 간격 & 반경

### Border Radius

| 토큰 | px | 용도 |
|---|---|---|
| `rounded-sm` | 2px | 뱃지, 작은 태그 |
| `rounded` | 3px | 버튼, 입력 필드 |
| `rounded-md` | 4px | 카드, 테이블 |
| `rounded-lg` | 6px | 모달, 큰 카드 |
| `rounded-full` | 9999px | 아바타, 카테고리 칩 |

### Shadow

| 토큰 | 정의 | 용도 |
|---|---|---|
| `shadow-none` | none | 기본 카드 |
| `shadow-sm` | `0 1px 2px rgba(0,0,0,0.04)` | 살짝 떠 있는 카드 (드물게) |
| `shadow-md` | `0 4px 12px rgba(0,0,0,0.06)` | 드롭다운, 팝오버 |
| `shadow-lg` | `0 8px 24px rgba(0,0,0,0.12)` | 모달 |
| `shadow-xl` | `0 20px 60px rgba(0,0,0,0.3)` | 로그인 카드 (어두운 배경 위) |

type BookStatus = 'available' | 'rented' | 'overdue' | 'disposed';

export function BookStatusBadge({ status, dueDate }: { 
  status: BookStatus; 
  dueDate?: string;
}) {
  const config = {
    available: { className: 'badge-available', label: '대여 가능' },
    rented: { className: 'badge-active', label: `~${dueDate}` },
    overdue: { className: 'badge-overdue', label: `연체 D+${calcDays(dueDate)}` },
    disposed: { className: 'badge-returned', label: '폐기' },
  };
  
  const { className, label } = config[status];
  return <span className={className}>{label}</span>;
}

### 5.2 KPI 카드 (대시보드)

```tsx
type Variant = 'default' | 'primary' | 'danger';

export function KpiCard({ 
  label, value, delta, variant = 'default' 
}: {
  label: string; 
  value: number | string; 
  delta?: string;
  variant?: Variant;
}) {
  const borderClass = {
    default: 'border-border',
    primary: 'border-t-foreground border-t-[3px]',
    danger: 'border-t-destructive border-t-[3px]',
  }[variant];
  
  return (
    <div className={`bg-card p-6 rounded-md border ${borderClass}`}>
      <div className="text-xs text-muted-foreground uppercase tracking-wider mb-3">
        {label}
      </div>
      <div className={`text-4xl font-bold tracking-tight tabular ${
        variant === 'danger' ? 'text-destructive' : ''
      }`}>
        {value}
      </div>
      {delta && (
        <div className="text-xs text-muted-foreground mt-1.5">{delta}</div>
      )}
    </div>
  );
}
```

## 6. 반응형 브레이크포인트

| 토큰 | min-width | 용도 |
|---|---|---|
| `sm` | 640px | 작은 태블릿, 큰 휴대폰 가로 |
| `md` | 768px | 태블릿 |
| `lg` | 1024px | 작은 데스크탑 |
| `xl` | 1280px | 일반 데스크탑 |
| `2xl` | 1400px | 큰 데스크탑 (관리자 1920px도 포함) |

**구성원 페이지 그리드 가이드**:
- 모바일 (< sm): 도서 카드 2열
- sm ~ md: 3열
- md 이상: 4열

**관리자 페이지**: 1280px 이상 가정. 그 이하에서는 가로 스크롤 허용.

- [ ] 액센트 색 위 텍스트는 항상 `text-primary-foreground`(흰색)
- [ ] 포커스 링은 `ring-ring` (`outline: 2px solid` 효과)
- [ ] 색만으로 정보 전달 금지 (예: 연체는 색 + "D+5" 텍스트)
- [ ] 키보드 네비게이션: shadcn/ui 기본 동작 활용
