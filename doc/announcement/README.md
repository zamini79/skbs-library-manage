# 사내 도서관 오픈 공지 자료

내일 전사 메일·게시판 공지에 사용할 자료 모음입니다. **앱 소스와 무관한 별도 자료**입니다.

## 파일

| 파일 | 용도 |
|---|---|
| `공지문.md` | 게시판/메일 본문 — 복사해서 붙여넣기 |
| `모바일_안내이미지.html` | 모바일 이용 안내 인포그래픽 — 이미지로 변환해 첨부 |
| `README.md` | (이 파일) 사용법 |

---

## 1) 공지문 사용
`공지문.md` 내용을 그대로 게시판/메일에 붙여넣으면 됩니다. 마크다운을 지원하지 않는
게시판이라면 제목·굵게 표시만 손봐서 올리세요.

---

## 2) 안내 이미지를 PNG로 만들기 (가장 쉬움)

`모바일_안내이미지.html` 을 **크롬/엣지에서 열고** 아래 중 하나로 캡처하세요.

**방법 A — 화면 캡처 (권장, 가장 빠름)**
1. HTML 파일을 더블클릭해 브라우저로 엽니다.
2. 가운데 카드(둥근 모서리 영역)만 화면에 꽉 차게 한 뒤
   - Mac: `Cmd+Shift+4` 로 카드 영역 드래그 캡처
   - Windows: `Win+Shift+S` (캡처 도구)로 카드 영역 드래그
3. 저장된 PNG를 공지에 첨부.

**방법 B — 전체 페이지를 깔끔하게 (개발자도구)**
1. 브라우저에서 HTML 열기 → `F12`(개발자도구) → `Cmd/Ctrl+Shift+P`
2. `Capture node screenshot` 입력 → 실행
3. Elements 탭에서 `<div class="card">` 요소를 선택한 상태로 실행하면
   카드만 깔끔하게 PNG로 저장됩니다. (여백/그림자 포함)

**방법 C — PDF로**
- 브라우저 인쇄(`Cmd/Ctrl+P`) → 대상 "PDF로 저장" → 배경 그래픽 켜기.

> 카드 고정 폭은 760px 입니다. 모바일 게시판에서도 선명하게 보입니다.
> 색상은 실제 앱과 동일한 서재 톤(브라운/크림)을 사용했습니다.

## 3) (선택) AI로 일러스트 버전을 만들고 싶다면

HTML 인포그래픽 대신 **그림 스타일 포스터**를 원하면, Claude.ai 또는
이미지 생성 도구에 아래 프롬프트를 넣어보세요.

```
사내 도서관 모바일 앱 오픈 안내 포스터를 만들어줘. 세로형(9:16).
- 따뜻한 서재 톤: 크림색 배경(#f5ecdc), 진한 브라운(#2a1d10), 포인트 brown(#7c3a1d)
- 상단: 책 이모지와 "SK bioscience 사내 도서관 OPEN" 타이틀
- 중앙: 스마트폰을 든 손, 화면에 책 표지 그리드가 보임
- 하단에 3단계 아이콘: ① 사내 이메일로 가입 ② 책 선택 후 '대출 신청' ③ 1층 안내데스크에서 수령
- 깔끔한 플랫 일러스트, 한글 텍스트, 미니멀
```

> 단, 생성형 이미지에 들어간 **한글 텍스트는 깨질 수 있으므로**, 글자가 중요한
> 공지에는 `모바일_안내이미지.html` 캡처본을 권장합니다.
